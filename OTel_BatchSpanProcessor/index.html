The maximum number of spans exported in one batch. Defaults to 512.

The maximum number of spans queued for export. Defaults to 2048. Old spans will be dropped if the number of queued spans exceeds this max size.
interval Time​Amount

The time interval in which to export batches. Defaults to 5 seconds.

The exporter which receives processed span batches.
