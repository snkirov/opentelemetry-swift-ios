Structure OTel.​Multiplex​Propagator 

public struct MultiplexPropagator: OTelPropagator

A pseudo-OTelPropagator that may be used to instrument using multiple other OTelPropagators across a common OTel.SpanContext.

Extraction Priority

The order of the given propagators matters when extracting! If multiple propagators successfully extract, the extracted span context of the last propagator called, i.e. the last propagator in the propagators array.
