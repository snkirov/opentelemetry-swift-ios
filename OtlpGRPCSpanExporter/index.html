Class Otlp​GRPCSpan​Exporter 

public final class OtlpGRPCSpanExporter: OTelSpanExporter

A span exporter which sends spans to an OTel collector via gRPC in the OpenTelemetry protocol (OTLP).

In order for this exporter to work you must have a running instance of the OTel collector deployed. Check out the 'OTel Collector: Getting Started' docs in case you don't have the collector running yet.
